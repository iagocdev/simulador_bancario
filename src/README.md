#  Simulador Bancário - Java Puro

Projeto acadêmico/pessoal desenvolvido em Java puro (Java SE) simulando um sistema de cadastros e operações bancárias do Banco Itaú.

## Evolução do Código
O objetivo principal deste repositório é registrar a minha evolução com a linguagem Java:
- **Fase 1:** Sistema simples com terminal controlado por loops e listas dinâmicas (`ArrayList<String>`).
- **Fase 2:** Evolução para Programação Orientada a Objetos (POO), encapsulamento de dados com a classe `Pessoa`.
- **Fase 3:** Implementação de conceitos de composição de objetos e regras de negócios com a classe `ContaBancaria` (operações de Saque e Depósito).

## Tecnologias Utilizadas
- Java 17+ (JDK)
- Biblioteca nativa `java.util.Scanner` e `java.util.ArrayList`
